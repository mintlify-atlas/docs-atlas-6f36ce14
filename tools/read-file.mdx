---
title: "read_file tool"
description: "Reads any file from the filesystem and returns its full contents as a plain string. Useful for inspecting source code, config files, or any text artifact."
---

The `read_file` tool reads a file at the given path and returns its entire contents as a plain string. It is backed by `os.ReadFile` and supports both absolute and relative paths.

## Arguments

<ParamField body="fileName" type="string" required>
  Path to the file to read. Accepts absolute paths (e.g., `/etc/hosts`) or paths relative to the working directory (e.g., `./main.go`).
</ParamField>

## Return value

On success, the tool returns the raw file contents as a string. On failure it returns an error message in the form:

```
ERROR: tools calling error: <message>
```

## Source

```go
var ReadFileTool = loop.NewSimpleTool(
    "Reads file from filesystem",
    []loop.ToolArgument{{Name: "fileName", Type: "string"}},
    func(args map[string]any) string {
        fileName, err := StringArg(args, "fileName")
        if err != nil {
            return "ERROR: tools calling error: " + err.Error()
        }

        dat, err := os.ReadFile(fileName)
        if err != nil {
            return "ERROR: tools calling error: " + err.Error()
        } else {
            return string(dat)
        }
    },
)
```
