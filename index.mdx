hrns (harness) is the simplest agent harness for Go. It gives you a streaming agent loop with tool support, a pluggable tool interface, a skills system for loading contextual instructions, and a terminal UI — all in one minimal framework. Wire up any OpenAI-compatible API endpoint and start building agents in minutes.

## How hrns works

hrns runs a streaming agentic loop: it sends your messages to an OpenAI-compatible LLM, streams the response, detects tool calls, executes the requested tools, appends results to the conversation, and repeats — until the model stops requesting tools.

<Steps>
  <Step title="Install hrns">
    Clone the repository and build the binary with `go build`.
  </Step>
  <Step title="Set environment variables">
    Export `HRNS_KEY`, `HRNS_BASE_URL`, and optionally `HRNS_SKIP_VERIFY`.
  </Step>
  <Step title="Run the TUI">
    Launch `./hrns` and start chatting. Use `/model` to switch models, `/new` for a fresh session.
  </Step>
  <Step title="Extend with custom tools">
    Implement the `loop.Tool` interface and pass your tools to `loop.New()`.
  </Step>
</Steps>
